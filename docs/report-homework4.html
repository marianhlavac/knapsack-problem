Jak číst v grafech¶

Data pro grafy byly ukldany automaticky, take jejich kivky jsou pojmenovny velmi zvltn. D se z nich vak velmi jednodue vyst nastaven parametr programu. Kad parametr je oznaen psmenem (nap. x pro crossover) a za nm nsleduje seln hodnota, kter je podle povahy parametru cel slo nebo slo s desetinnou st.

Cel etzec tak oznauje nastaven:

xXXmXXeXtXpX

  • x pravdpodobnost kzen (00 - 09, 0.0 a 0.9)
  • m pravdpodobnost mutace (00 - 09, 0.0 a 0.9)
  • e elitismus (velikost penesen populace do nov)
  • t poet turnaj
  • p velikost turnaje

def read_results(pattern):
    data = pd.DataFrame({})
    for file in glob.glob(pattern):
        raw = pd.read_csv(file)
        if (data.empty):
            data = raw
        else:
            data = data.merge(raw, left_index=True, right_index=True)
    return data
